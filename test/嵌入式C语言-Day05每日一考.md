##### 1 【问答题】 break 和 continue 有什么区别

```
break 跳出整个循环
continue 跳出本次循环，下一次继续
```

##### 2 【问答题】 while 和 do while 有什么区别

```
while: 每次循环都是先判断再执行循环体语句
do while： 第一次没有判断直接执行循环体语句，但以后每一次执行也经过判断
```

##### 3.【代码题】请写出下面程序的运行结果

```c
int sum = 0;
for (int i = 1; i < 5; i++)
{
    if (i == 4)
    {
        continue;
    }
    sum += i; 
}
printf("%d", sum);
```

```
6
```

##### 4.【代码题】以下代码共输出多少个”尚硅谷“

```c
for (int i = 1; i <= 3; i++)
{
    for (int j = 1; j <= i; j++)
    {
        printf("尚硅谷 ");
    }
}
```

```
i=1, 内循环1次
i=2，内循环2次
i=3，内循环3次
```

```
6个
```
